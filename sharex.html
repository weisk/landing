owo.sxcu

To generate a ShareX .sxcu uploader configuration for whats-th.is, simply insert your user token into the text field below and press the submit button to start the download. Once the file is downloaded, simply click it to prompt ShareX to configure itself to use owo.

Note: your token is not transmitted over the wire by this tool, and configuration files are generated in your browser using Javascript.
